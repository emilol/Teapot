Teapot Resharper Plugin

A navigation extension that filters implementations of generic interfaces by type parameters

Download Teapot from the Resharper Gallery

Navigating generic types in Resharper is a pain. When Resharper 9 came along and didn't have much to say about the problem, we had all the impetus we needed to solve it ourselves.

Without Teapot

With Teapot

Implementation results are filtered to only include those with matching type parameters. If the codebase only has one implementation that matches, you're taken straight to it.
